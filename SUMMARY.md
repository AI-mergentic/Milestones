# Milestones (v1.19.x)

An optimized campaign development template for Battle for Wesnoth.

## Features
- **Centralized Roles**: Manage characters and recruits in `lua/dramatispersonae.lua`.
- **Dynamic WML**: Automatic synchronization between Lua logic and WML scenarios.
- **Abstract Assets**: Integrated maps and terrains from high-quality UMC sources.
- **Scaffolded Scenarios**: Use `00_Template.cfg` to start coding faster.
